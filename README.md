# frontend-web

## Project setup
```
yarn install
```

Copy the file .env.template to .env in the root project

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Run your unit tests
```
yarn test:unit
```

### Lints and fixes files
```
yarn lint
```
